# Design Check List

- [ ] 设计尺寸和显示尺寸有差异的设计图比例问题
- [ ] 考虑不同尺寸window显示状况
- [ ] UI是否有矛盾
- [ ] 少404、500等特殊页面
- [ ] 用户登录与未登录的状态是否完整
- [ ] 列表各类loading效果有没有指定
- [ ] 数据最多和最少的情况怎么展示
- [ ] 表单有没有多次提交的验证码
- [ ] 字体大小限定：中文最小12px 日文/英文10px
- [ ] 字体可不可以使用
- [ ] Hover效果是否有提供
- [ ] Active效果是否有提供
- [ ] 动画效果是否有提供
- [ ] 考虑实际内容的显示 
- [ ] SP的内容一定左右有padding（8/12/16px）
- [ ] button height 统一 PC：44/36 SP: 36/32
